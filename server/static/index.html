Worker komutu

Her cihazda kendi localllm klasöründe çalıştır. Komut ayar dosyasını yazar ve worker'ı eğitim/senkron işleri açık şekilde başlatır.

python scripts/setup_worker.py --server http://PANEL_IP:PORT --name my-node --token API_TOKEN
